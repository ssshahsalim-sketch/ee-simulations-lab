HOW TO DEPLOY ON RENDER.COM
============================

STEP 1 - Create GitHub Account
--------------------------------
1. Go to https://github.com
2. Sign up for free
3. Verify your email

STEP 2 - Create a New Repository
----------------------------------
1. Click the "+" icon (top right) → "New repository"
2. Name it: ee-simulations-lab
3. Set to PUBLIC
4. Click "Create repository"

STEP 3 - Upload Your Files to GitHub
--------------------------------------
1. On your new repo page, click "uploading an existing file"
2. Drag and drop ALL files from your ee_website folder:
   - app.py
   - requirements.txt
   - render.yaml
   - meta.json (if exists)
   - content.json (if exists)
3. Also drag the folders:
   - templates/ (with all 3 html files)
   - static/
   - simulations/ (with your html files)
4. Click "Commit changes"

STEP 4 - Deploy on Render
---------------------------
1. Go to https://render.com
2. Sign up with your GitHub account
3. Click "New +" → "Web Service"
4. Click "Connect" next to your ee-simulations-lab repo
5. Fill in:
   - Name: ee-simulations-lab
   - Runtime: Python
   - Build Command: pip install -r requirements.txt
   - Start Command: gunicorn app:app
6. Click "Create Web Service"
7. Wait 2-3 minutes for deployment
8. Your site is LIVE at: https://ee-simulations-lab.onrender.com

DONE! Share that URL with anyone in the world!
