Protection Function Execution Order
Each cycle, the relay evaluates functions top-to-bottom against live CT/PT samples. A function that reaches pickup starts its timer; reaching the set time (or instantaneous threshold) issues a trip command to the output relay driving the breaker trip coil.
Protection Settings
Settings apply live to the running simulation and to the Testing / Calculators modules. Values follow typical IEC 60255-8 thermal and inverse-time overcurrent conventions for demonstration purposes.
