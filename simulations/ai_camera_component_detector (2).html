How This Tool Works

1. Live camera preview runs entirely in your browser using the standard getUserMedia API — no video is transmitted anywhere for the preview itself.

2. AI analysis happens only when you click "Analyze Frame with AI" (or when an auto-analyze interval fires): a single JPEG snapshot of the current frame is sent directly from your browser to Anthropic's Claude API for identification, using your own API key, and returns a structured description — including electrical-engineering-specific details when the object is recognized as electrical/electronic equipment.

3. Nothing is stored server-side by this tool. The capture history strip lives only in this browser tab's memory and disappears when you reload or close the page.

Why an API key is needed

This page is a standalone file you run directly in your own browser (required for reliable camera access). Because of that, it can't use Anthropic's built-in chat credentials the way the Claude.ai website does — it needs to authenticate its own API calls. Paste a personal API key from console.anthropic.com/settings/keys into the "Anthropic API Key" field on the Live Detection tab. The key is sent only directly to Anthropic's servers with each analysis request (via Anthropic's documented direct-browser-access header) and is never sent anywhere else. Optionally tick "Remember in this browser" to save it in this browser's local storage for next time — untick/clear it on a shared computer.

Requirements

Camera access needs your browser's permission — if it doesn't prompt automatically, check the camera icon in your address bar or your OS privacy settings. The AI analysis feature (not the camera preview itself) requires an active internet connection and a valid Anthropic API key, since each "Analyze" click calls the Claude API directly and uses a small amount of your API credit.

Accuracy Disclaimer

AI-based visual identification is a convenience aid for training and quick reference — it can occasionally misidentify look-alike components (for example a contactor vs. a relay, or an MCB vs. an MCCB at odd angles) and cannot read fine nameplate text reliably from a distance. Always confirm ratings, wiring, and component identity from the physical nameplate and manufacturer documentation before making any engineering or safety decision.
