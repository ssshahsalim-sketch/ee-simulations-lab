🚀 How to get TRUE background monitoring

A
Quick (5 min) — host free on GitHub Pages: Create a free GitHub account → New repository → Upload this HTML as index.html → Settings → Pages → Enable. You get a free HTTPS URL like yourname.github.io/ip-monitor. Open that URL on your phone in Chrome, tap Add to Home Screen. The installed PWA then supports background periodic checks on Android Chrome.
B
Best (Android) — Termux background script: Install Termux from F-Droid → run pkg install python termux-api → run the Python script below. It checks your IP every 5 minutes and sends a real Android notification even with the screen off, forever.
# Save as ip_monitor.py in Termux, then run:
# python ip_monitor.py &
# (the & runs it in background)

import subprocess, time, json, urllib.request

INTERVAL = 300  # seconds between checks
last_public = None

def get_public_ip():
    try:
        with urllib.request.urlopen(
            'https://api.ipify.org?format=json', timeout=8) as r:
            return json.load(r)['ip']
    except:
        return None

def notify(title, msg):
    subprocess.run(['termux-notification',
        '--title', title, '--content', msg,
        '--priority', 'high'])

print('IP Monitor started.')
while True:
    ip = get_public_ip()
    if ip and ip != last_public:
        if last_public:
            notify('IP Changed!',
                f'Was: {last_public}\nNow: {ip}')
        last_public = ip
        print(f'IP: {ip}')
    time.sleep(INTERVAL)
