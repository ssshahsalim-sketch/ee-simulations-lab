Rungs scan top→bottom, every cycle. COIL follows the rung live. SET/RESET latch a bit until reset/set elsewhere — use SET/RESET on breaker tags for real trip/close commands. A device tag only responds to PLC coils while its mode is AUTO (toggle mode in the Tags tab or on the diagram).
How this works

This is a simplified 33kV/11kV substation: Incomer → Isolator/Breaker/Isolator → Bus‑1 → Transformer bay → Bus‑2 → 3 outgoing feeders.

SCADA (right side)
Click any breaker (rectangle) or isolator (diagonal blade) to open/close it, if that device is in MANUAL mode. Isolators are hard‑interlocked: you cannot open/close an isolator while its adjacent breaker is CLOSED (standard substation safety practice) — this mirrors real switching sequences.

PLC (left side)
Build ladder rungs from contacts (NO/NC) of any tag, driving a COIL / SET / RESET on an output tag. Put a device in AUTO mode and its position will be fully driven by your program — e.g. trip a feeder breaker automatically on an overcurrent bit. Press "Start PLC Scan" to run the program continuously (500 ms scan).

Simulation engine
Every scan, power flow is recalculated: a bus is "live" only if every switching device between it and the source is CLOSED. Live buses/feeders get realistic V / I / P / F readings with small jitter; de‑energized points read 0. Use the Fault panel (bottom right) to spike a feeder's current and test your protection logic.
