Earth Electrode ResistanceSingle rod resistance, combined into a grid of N electrodes
Formula
Rₑ = (100ρ / 2πL) × ln(2L / d)
Rₑ = resistance of one electrode (Ω)  ·  ρ = soil resistivity (Ω·m)  ·  L = electrode length (cm)  ·  d = electrode diameter (cm)
Single electrode, Rₑ 8.3367 Ω
Combined, R_earth = Rₑ / N 0.4168 Ω
The simple Rₑ/N rule (as used in the reference sheet) assumes fully independent, widely-spaced rods. In a real grid the electrodes are close enough to interact ("mutual resistance"), so actual combined resistance is typically 15–40% higher than this ideal value. Use IEEE 80 grid formulas or measured fall-of-potential tests to confirm before finalising a critical design.

Tolerable limits (50 kg body weight)
E_touch50 = (1000 + 1.5·Cs·ρs) × 0.116/√ts

Cs = surface derating factor (≈1 if no crushed-rock layer)  ·  ρs = surface layer resistivity (Ω·m)  ·  ts = shock duration (s), taken = fault duration t
0 = no gravel/crushed-stone layer (bare soil). 2000–3000 typical for 100–150mm crushed stone.
Cs = 1 if no derating layer used. Typically 0.6–0.8 with a crushed-stone surface.
