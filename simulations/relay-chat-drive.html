Relay // One-time setup
Connect Google Drive
Relay stores every message and file in a Google Drive folder you control. This needs a free OAuth Client ID from your own Google Cloud project — takes about 3 minutes, once.
How to get a Client ID:
  1. Go to console.cloud.google.com/apis/credentials
  2. Create a project (or pick one), then enable the Google Drive API
  3. Create credentials → OAuth client ID → Application type: Web application
  4. Under "Authorized JavaScript origins" add the exact URL you're serving this file from (e.g. http://localhost:8000 or your Render/GitHub Pages domain) — not a file:// path
  5. Copy the generated Client ID and paste it below
Stored only in this browser's local storage. Never sent anywhere except Google.
